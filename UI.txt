## 关键优化点

### 1. 统计卡片（顶部）
- 四个卡片等宽，使用图标+数字+标签的垂直布局
- "等待反馈"有数据时显示脉冲动画/高亮边框，并显示[催办]快捷按钮
- 数字字体加大加粗（32px），状态颜色柔和化（不用纯红，用橙红）
- 当前有数据的卡片轻微上浮阴影，吸引注意力

### 2. 智能解析区域（可折叠）
- 默认折叠，显示"✨ 粘贴描述自动解析"提示条
- 点击展开文本域，解析成功后自动填充下方表单并收起
- 解析失败时不阻断用户，允许直接手动填写

### 3. 快速记录表单（左栏）
- 客户输入框支持实时搜索下拉（显示客户名+地区+联系人）
- 选中客户后，输入框右侧显示✓和"已存在客户"标签
- 未选中客户时，回车直接创建新客户（弹窗确认）
- 类型选择后，标题输入框placeholder动态变化：
  - 硬件故障 → "例如：相机无法开机/连接超时..."
  - 软件使用 → "例如：SDK调用返回错误码..."
  - 咨询 → "例如：咨询保修政策/升级方案..."
- 描述文本域高度自适应（最小3行，最大6行）
- 底部操作栏固定在表单底部（不随滚动消失）
- 保存成功后：
  - 显示"✓ 已保存 #2024"提示
  - 表单重置，客户输入框聚焦（连续录入场景）

### 4. 待办列表（右栏）
- 工单卡片式布局（替代表格），每行显示：
  - 左侧：优先级色块（紧急红/高橙/普通灰）
  - 中间：标题（单行截断）、客户名、负责人、时间
  - 右侧：[处理]按钮（点击进入工单详情）
- 超过24小时的工单左侧加🔥标记
- 等待客户反馈的工单显示⏳和"已等待X天"
- 底部固定显示今日统计小卡片

### 5. 空状态优化
- 待办为空时显示："🎉 今日待办已清空，喝杯咖啡吧" + 快捷创建按钮
- 统计为0时数字显示灰色（不是黑色），避免视觉噪音

### 6. 响应式规则
- 宽屏（>1400px）：三栏（统计+表单+待办）
- 中屏（>1024px）：两栏（统计/表单+待办）
- 平板（<1024px）：单栏，待办折叠到下方
- 手机（<768px）：单栏，统计横向滚动，表单全宽

## 交互细节

### 客户搜索下拉
输入"广州" → 下拉显示：
┌─────────────────────────┐
│ 广州图博士      广州｜王工 │
│ 广州视觉科技    广州｜李经理│
│ ───────────────────────── │
│ [+] 创建新客户：广州...    │
└─────────────────────────┘

### 保存反馈
保存成功 → 右下角滑入提示：
┌─────────────────┐
│ ✓ 工单 #1245 已创建    │
│ [查看] [再记一条]      │
└─────────────────┘
3秒后自动消失，或鼠标悬停暂停

点击"催办" → 弹出选择框：
┌─────────────────┐
│ 催办方式：        │
│ ○ 系统内通知      │
│ ○ 复制催办文案    │
│   "您好，关于XX问题..."│
│ [确认催办]        │
└─────────────────┘
