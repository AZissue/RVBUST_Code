# 编码圆拼接工具 v2

基于 RVC 深度相机的编码圆点云拼接系统，支持多帧点云自动配准与彩色融合。

## ✨ 主要特性

- **🔧 工程化设计** - 模块化架构，状态管理清晰，资源安全释放
- **📷 实时预览** - 2D 实时视频流 + 编码圆实时检测
- **🎨 彩色点云** - SDK 原生颜色映射，避免手动投影错误
- **💾 工程文件** - 支持 `.ccproj` 格式保存/加载，位姿矩阵持久化
- **🔄 帧管理** - 支持单帧删除、上移、下移、重新拼接
- **📊 位姿导出** - 拼接结果可导出 JSON 格式位姿矩阵
- **🔍 参数搜索** - 自动搜索最佳编码圆检测参数

## 🚀 快速开始

### 环境要求

- Python 3.10+
- PySide6
- OpenCV
- NumPy
- Open3D（可选，用于点云预览）
- RVC SDK（已安装相机驱动）

### 安装依赖

```bash
conda create -n rvc python=3.10
conda activate rvc
pip install PySide6 opencv-python numpy open3d
```

### 运行程序

```bash
conda activate rvc
python src/main.py
```

## 📖 使用指南

### 1. 连接相机
- 点击「🔍 查找设备」确认相机被识别
- 点击「连接」建立连接
- 状态栏显示相机型号和类型（X1/X2）

### 2. 拍摄参数设置
- **2D 曝光** - 影响实时预览亮度（支持实时调节）
- **3D 曝光** - 影响结构光拍摄质量
- **2D/3D 增益** - 调节传感器增益
- 点击「应用并保存」持久化到相机

### 3. 编码圆参数
- **N 值** - 编码圆的总数量（如 8、12）
- **r1/r0** - 内环半径比（建议 1.5-2.5）
- **r2/r0** - 外环半径比（必须 > r1/r0）
- 输入错误时会自动禁用拍摄按钮

### 4. 拍摄与拼接
1. 点击「📷 3D 拍照」获取点云
2. 确认当前帧检测到足够编码圆（建议 ≥3 个）
3. 点击「➕ 添加」加入拼接队列
4. 重复步骤 1-3 添加多帧
5. 点击「🔗 执行拼接」计算位姿
6. 拼接成功后可用 Open3D 预览

### 5. 帧管理
- 右键拼接帧列表可删除/上移/下移
- 支持重新调整顺序后重新拼接
- 可导出位姿矩阵到 JSON

### 6. 工程文件
- 「💾 保存工程」- 保存所有帧数据、位姿、参数
- 「📂 加载工程」- 恢复之前的工作状态
- 工程文件为 `.ccproj` 格式（实际为 ZIP）

## ⚠️ 常见问题

### 1. 编码圆检测不到
- 检查 N 值是否与实际编码圆数量匹配
- 尝试调整 r1/r0 和 r2/r0 比值
- 高分辨率相机建议使用更小的比值
- 点击「🔍 测试当前帧检测」自动搜索最佳参数

### 2. 拼接失败
- 确保相邻帧有 ≥3 个共有编码圆
- 检查编码圆是否在视场中央且清晰
- 避免拍摄过程中物体移动
- 查看日志中的具体错误码描述

### 3. Open3D 预览黑屏
- 已自动过滤 NaN/Inf 无效点
- 确保已安装 Open3D：`pip install open3d`

### 4. 相机连接失败
- 检查相机电源和 USB 连接
- 确认 RVC SDK 驱动已安装
- 尝试重新插拔相机
